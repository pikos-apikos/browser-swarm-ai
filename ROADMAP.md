# Roadmap

## Phase 0
- write the minimal spec
- define manifest schema
- define peer lifecycle
- pick one demo runtime target

## Phase 1
- implement tracker
- implement browser swarm client
- implement chunk verification
- implement local cache

## Phase 2
- load a small model in-browser
- connect swarm delivery to runtime load
- expose a minimal chat demo

## Phase 3
- add peer capability advertisement
- add optional coordinator stub
- add task offer and claim flow

## Phase 4
- add verification hooks
- experiment with shared batch inference
- document threat model and privacy tradeoffs
